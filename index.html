ASP

Agent Sharing Protocol

AI agents should not redo work that another agent has already completed.

ASP is an open protocol that allows AI agents to share completed work results as reusable artifacts.

Instead of sharing abilities or skills, agents share verifiable outcomes — analysis, audits, research, reasoning, and computation results — so other agents can reuse them without recomputing the original task.

ASP is an open, permissionless protocol for agent-to-agent collaboration.
